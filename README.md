## 开始使用
1.添加css样式
    使用前请在css中开启border-box
```css
*,*::before,*::after{box-sizing: border-box;}
```
    
还需要设置默认的颜色（由于css变量兼容性问题，后续会改进scss）

```css
    :root {
    --font-size: 14px;
    --button-height: 32px;
    --button-bg: white;
    --button-active-bg: #eee;
    --border-radius: 4px;
    --color: #333;
    --border-color: #999;
    --border-color-hover: #666;
    }
```
