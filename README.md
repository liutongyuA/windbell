# 风铃UI组件库

## 介绍
一个UI组件库，是我学习Vue做的项目

作者：liutongyua

## 开始使用
1.添加css样式
    使用前请在css中开启border-box
```css
*,*::before,*::after{box-sizing: border-box;}
```
2.安装 windbell
```
npm i windbell / yarn add windbell
```

3.引入
```js
import * as wb from 'windbell'
import 'windbell/dist/windbell.css'
```
